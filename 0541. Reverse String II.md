## Initial Solution
1. Use `head` and `tail` pointers to define the start and end boundaries of target substring to be reversed.
2. Iterate through the string in the chunks of `2*k`, using `left` and `right` pointers to perform aan in-place reversal on the the substring.

```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        int head = 0, tail = k-1;
        while(head < s.size()){
            int left = head, right = tail;
            if (tail >= s.size())
                right = s.size()-1;
            while(left < right){
                swap(s[left++],s[right--]);
            }
            head += 2*k;
            tail += 2*k;
        }
        return s;
    }
};
```

## Note
- Time complexity: $O(n)$
- Space complexity: $O(1)$
