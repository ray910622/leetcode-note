## Initial Solution 
```cpp
class Solution {
public:
    string removeDuplicates(string s) {
        string res;
        for (char c : s){
            if (!res.empty() && res.back() == c){
                res.pop_back();
            }
            else res.push_back(c);
        }
        return res;
    }
};
```

## Note 
- Time Complexity: $O(n)$
- Space Complexity: $O(n)$
